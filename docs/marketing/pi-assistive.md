# PI Is Assistive, Not Authoritative

**URL:** `/pi-assistive`

Public positioning page explaining PI as an assist plane. PI may draft, summarize, and explain, but it does not own routing states, final dispositions, approvals, sends, thresholds, or watchlist content.

PI and Sentinel can surface operational evidence for operator attention, including queue summaries and readiness context. Public copy must make clear that this visibility is not autonomous execution.

Claim boundary: do not imply PI independently determines compliance outcomes.
